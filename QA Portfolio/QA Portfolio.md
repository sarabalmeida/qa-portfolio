# 👋 Bem-vindo!

Este repositório foi criado para demonstrar minha experiência e evolução na área de **Qualidade de Software (QA)**.

Aqui você encontrará exemplos de artefatos utilizados durante todo o ciclo de testes de software, desenvolvidos para fins de estudo e demonstração profissional, sempre preservando a confidencialidade de projetos reais.

## 👩‍💻 Sobre mim

Sou **Analista de Qualidade de Software (QA)** com experiência desde **2018**, atuando em projetos de:

* 🏥 Saúde
* 🏛️ Administração Pública
* 💼 ERP
* 💳 Sistemas Financeiros
* 🔗 APIs REST

Tenho experiência com testes funcionais, exploratórios, regressivos, integração, APIs, banco de dados, documentação de testes e acompanhamento do ciclo completo de desenvolvimento.
